Tic-Tac-Toe v. 1.0
By: Christopher Segale

Requirements for compilation: asdf (for library management), lispbuilder-sdl, lispbuilder-sdl-ttf, lispbuilder-sdl-image

Instructions: Players click the squares on the grid in order to mark the squares.  Player one moves first while player two moves second.  The game is won when three identical marks align vertically, horizontally, or diagonally.
